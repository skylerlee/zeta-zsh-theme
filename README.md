# Zeta ZSH Theme

Another :star:theme for oh-my-zsh

## Screenshot
![screenshot](screenshot.png)

## Features
* User name
* Machine name
* Current working directory
* Git branch
* Git status
    * `✔` —— Clean branch
    * `✘` —— Dirty branch
    * `+` —— Added files
    * `-` —— Deleted files
    * `*` —— Modified files
    * `>` —— Renamed files
    * `=` —— Unmerged changes
    * `?` —— Untracked changes
* Prompt indicator turns red if the last run fails
